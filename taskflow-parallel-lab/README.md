## Quick Reference — What Each File Does

### Mappings
```
M_Load_Customers  → 20 customer rows → TGT_CUSTOMERS.csv
M_Load_Products   → 20 product rows  → TGT_PRODUCTS.csv
M_Load_Orders     → 30 order rows    → TGT_ORDERS.csv
M_Load_Sales_Fact → 50 fact rows     → TGT_SALES_FACT.csv
```

### Taskflow Flow
```
Start → Init_Variables → Fork Gateway
  ├─ Branch 1: MT_Load_Customers (parallel)
  ├─ Branch 2: MT_Load_Products (parallel)
  └─ Branch 3: MT_Load_Orders (parallel)
       └─→ Join Gateway (wait for all 3)
           └─→ MT_Load_Sales_Fact (sequential)
               └─→ Decision (check status)
                   └─→ End
```
